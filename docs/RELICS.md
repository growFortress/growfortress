# Relics

Relics are powerful items obtained after each wave. Choose 1 of 3 offered relics.

## Overview

~25 relics across 7 categories:

| Category | Count | Description |
|----------|-------|-------------|
| Build-Defining | 4 | Shapes entire playstyle |
| Standard | 5 | Basic stat bonuses |
| Class | 5 | Class-specific bonuses |
| Pillar | 3 | Pillar-specific bonuses |
| Synergy | 2 | Multi-system bonuses |
| Economy | 3 | Gold/dust bonuses |
| Cursed | 3 | High risk, high reward |

---

## Build-Defining Relics

These relics fundamentally change your build strategy.

### Splash Master
- **Rarity**: Epic
- **Effect**: Attacks deal 35% damage to nearby enemies
- **Synergy**: Fire class, Cannon Tower

### Chain Lightning
- **Rarity**: Epic
- **Effect**: +40% chain chance, +2 chain targets
- **Synergy**: Lightning class, Tesla Tower

### Executioner
- **Rarity**: Epic
- **Effect**: +150% damage to enemies below 25% HP
- **Synergy**: High attack speed builds

### Critical Eye
- **Rarity**: Epic
- **Effect**: +20% crit chance, +75% crit damage
- **Synergy**: Ice class, high damage builds

|--------|-------------|------------|

## Relic Selection Strategy

### Early Game (Waves 1-3)
- Prioritize damage relics
- Fortress Shield for survivability
- Avoid cursed relics

### Mid Game (Waves 4-7)
- Look for build-defining relics
- Class relics if available
- Economy relics for late-game scaling

### Late Game (Waves 8-10)
- Stack synergy relics
- Consider cursed relics for burst
- Pillar relics for boss fights

---

## Reroll System

- **Cost**: 1 reroll per wave (free)
- **Additional rerolls**: 50 gold each
- **Reroll pool**: Same rarity weights, different relics
