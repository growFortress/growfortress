# Pillars (World Sectors)

## Overview

The game features 6 pillars (world sectors), each with unique themes, enemies, bosses, and modifiers. Each pillar spans 10 waves.

---

## Pillar Progression

| Pillar | Waves | Element | Theme |
|--------|-------|---------|-------|
| Streets | 1-10 | Tech | Urban crime |
| Science | 11-25 | Lightning | Technology |
| Mutants | 26-40 | Cosmic | Mutant hunting |
| Cosmos | 41-60 | Cosmic | Space aliens |
| Magic | 61-80 | Void | Dark dimensions |
| Gods | 81-100 | Fire | Divine beings |

## Endless Mode Cycling

After wave 100, pillars cycle with increased difficulty:

```
cycle = floor((wave - 1) / 100)
effectiveWave = ((wave - 1) % 100) + 1
pillarIndex = floor((effectiveWave - 1) / 10)

cycleScale = 1.6^cycle
```

### Cycle Scaling
| Cycle | Waves | Stats Multiplier |
|-------|-------|------------------|
| 0 | 1-100 | 1.0× |
| 1 | 101-200 | 1.6× |
| 2 | 201-300 | 2.56× |
| 3 | 301-400 | 4.1× |
| 4 | 401-500 | 6.55× |

## Natural Hero Bonus

When using Natural class heroes, regardless of pillar:
- +20% damage multiplier
- +50% XP multiplier
